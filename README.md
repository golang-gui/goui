## Why goui?

### 🚀 Goroutine-Powered Concurrency
- **No more UI freezing**: All event handlers run in isolated goroutines
- **Concurrent-safe state management**: Built-in atomic state operations
- **Deadlock-free architecture**: Automatic synchronization between UI and business logic

### ✨ Pure Go Ecosystem
- **Zero CGO**: 100% Go implementation - no external dependencies
- **Single-binary deployment**: Compile self-contained executables
- **Cross-platform**: Windows, macOS, Linux support  (ing...)

### ⚡️ Modern Architecture
- **OpenGL rendering**: Hardware-accelerated graphics
- **Declarative value semantics**: Build UIs with pure Go constructs
- **Reactive state system**: Automatic UI updates on state changes
